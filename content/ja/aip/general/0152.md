# ジョブ

APIは、完了にかなりの時間がかかり、かつ一時的な[long-running operation][aip-151]が適切でないタスクを公開する必要がある場合がある。例えば、タスクを繰り返し実行する必要があったり、タスクの構成と実行に別々の権限が必要だったりする場合である。

## ガイダンス

APIは、個別のセットアップ、構成、および実行を持つ特定のタスクを表すために`Job`リソースを定義しても**してもよい**（may）：

```proto
message WriteBookJob {
  option (google.api.resource) = {
    type: "library.googleapis.com/WriteBookJob"
    pattern: "publishers/{publisher}/writeBookJobs/{write_book_job}"
  };

  // Name and other fields...
}
```

- リソースの名前は単語"Job"で終わら**なければならない**（must）。
  - プレフィックスは、動詞と名詞を持つ有効なRPC名と**すべきである**（should）。
- サービスは、5つの標準メソッド（AIP-131、AIP-132、AIP-133、AIP-134、AIP-135）をすべて定義し、それらをジョブを構成する主な方法として使用**すべきである**（should）。

### Runメソッド

サービスは、ジョブを即座に実行する`Run`カスタムメソッドを定義する**すべきである**（should）：

```proto
rpc RunWriteBookJob(RunWriteBookJobRequest)
    returns (google.longrunning.Operation) {
  option (google.api.http) = {
    post: "/v1/{name=publishers/*/writeBookJobs/*}:run"
    body: "*"
  };
  option (google.longrunning.operation_info) = {
    response_type: "RunWriteBookJobResponse"
    metadata_type: "RunWriteBookJobMetadata"
  };
}
```

- RPC名は単語`Run`で始まら**なければならない**（must）。RPC名の残りの部分は、実行されるジョブリソースの単数形と**すべきである**（should）。
- リクエストメッセージはRPC名と一致し、`Request`接尾辞が付か**なければならない**（must）。
- メソッドは[long-running operation][aip-151]を返す**べきである**（should）。これはジョブ実行の結果を含むレスポンスメッセージに解決**しなければならない**（must）。
  - レスポンスメッセージ名はRPC名と一致し、`Response`接尾辞が付か**なければならない**（must）。
  - メソッドは任意のメタデータメッセージを使用しても**してもよい**（may）。
- HTTP動詞は、[カスタムメソッド][aip-136]の通常通り、`POST`と**しなければならない**（must）。
- `google.api.http`アノテーション内のbody句は`"*"`と**すべきである**（should）。
- URIパスは、実行されるジョブリソースの名前に対応する単一の`name`変数を含む**べきである**（should）。
- URIパスは`:run`で終わら**なければならない**（must）。
- ジョブの実行開始を妨げるエラーは、他のメソッドと同様に、エラーレスポンス（AIP-193）を返す**しなければならない**（must）。ジョブ実行の過程で発生するエラーは、メタデータメッセージに配置して**もよい**（may）。エラー自体は依然として[`google.rpc.Status`][status]オブジェクトで表現され**なければならない**（must）。

### Runリクエストメッセージ

Runメソッドは共通のリクエストメッセージパターンを実装する：

```proto
message RunWriteBookJobRequest {
  // The name of the job to run.
  string name = 1 [
    (google.api.field_behavior) = REQUIRED,
    (google.api.resource_reference) = {
      type: "library.googleapis.com/WriteBookJob"
    }];
}
```

- 単一の`string name`フィールドを含め**なければならない**（must）。
  - フィールドは[requiredとしてアノテーションされる][aip-203]**べきである**（should）。
  - フィールドは参照する[リソースタイプ][aip-123]を識別**すべきである**（should）。

### 実行と結果

通常、APIは`Run`メソッドの最終レスポンスとしてユーザーに結果を提供する**すべきである**（should）。しかし、これでは不十分な場合もある。例えば、バックグラウンドで定期的に実行されるジョブは、この方法でユーザーに結果を配信できない。

サービスは、個々の実行をその結果とともにジョブの下のサブコレクションリソースとして保存しても**してもよい**（may）。これにより、ユーザーは過去のジョブ実行を一覧表示できる。これを行うサービスは、実行リソースに対して`Get`、`List`、および`Delete`メソッドを定義する**すべきである**（should）：

```proto
message WriteBookJobExecution {
  option (google.api.resource) = {
    type: "library.googleapis.com/WriteBookJobExecution"
    pattern: "publishers/{publisher}/writeBookJobs/{write_book_job}/executions/{execution}"
  };

  // Name and other information about the execution, such as metadata, the
  // result, error information, etc.
}
```

この場合、ジョブの`Run`メソッドによって返される操作は、子リソースを参照する**すべきである**（should）。
