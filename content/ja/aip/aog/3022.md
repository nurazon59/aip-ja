# Built-in intent引数と履行API

Actions on Googleの[built-in intent][bii]（BII）は_履行スキーマ_であり、Schema.orgを外部ボキャブラリとして使用する。BIIは_履行APIではなく_、[AIP-3021][]で定義されるActions on Google共通型を使用しない。

Actions on Googleには2種類の汎用履行APIがある：

- [App Actions][]のようなURIベースのAPI。文字列引数のみをサポートする。
- Google標準API。`string` のようなJSONプリミティブ型とActions on Google共通型をサポートする。

これらのAPIはSchema.org JSON-LDオブジェクトをサポートしない。これらのAPIは、BII引数またはそのプロパティを、自身が_サポートする_引数型スキーマにマッピングすることで、BII引数をサポートする**べきである**（should）。例えば、[MonetaryAmount][schema-monetaryamount] 型のBII引数は、[Money][] 型のAPIフィールドにマッピングされる可能性がある。

## BII引数パス

特定のSchema.orgエンティティ型のみが、特定の共通型にマッピングされるべきである。適切な型にマッピングできないBII引数をサポートするために、APIはBII引数パスによる構成をサポートしても**よい**（may）。BII引数パスは、適切な型にマッピング_できる_ Schema.orgエンティティのプロパティをユーザーが指定できるようにする文字列である。

BII引数パスをサポートするAPIは、このセクションで定義される構文を使用する**べきである**（should）。

BII引数パスはドット区切りの文字列であり、BIIへの引数リスト内の特定のSchema.orgエンティティを識別する。

例えば、以下の架空のBII引数 `grant` と `date` を考える：

```json
{
  "grant": {
    "@context": "http://schema.org",
    "@type": "MonetaryGrant",
    "funder": {
      "@type": "Organization",
      "name": "Google"
    },
    "amount": {
      "@type": "MonetaryAmount",
      "currency": "USD",
      "value": 100
    }
  },
  "date": "2000-01-01"
}
```

`MonetaryGrant` をマッピングできる共通型は存在しないが、`grant` 引数には [MonetaryAmount][schema-monetaryamount] 型の `amount` プロパティが含まれており、これは [Money][] にマッピング_できる_。このプロパティはパス文字列 `"grant.amount"` で参照できる。同様に、`grant.name` は文字列フィールドにマッピングできる。

[Money][] をサポートしないURIベースのAPIでは、代わりにパス文字列 `"grant.amount.currency"` と `"grant.amount.value"` を使用して、`grant.amount` を2つの個別のURIパラメータにマッピングする必要がある。

一方、`date` は [DateTime][] にマッピングできるため、パス文字列 `"date"` はURIベースのAPIとJSON APIの文字列フィールドの両方で十分である。

### 複数の値を持つプロパティ

任意のSchema.orgプロパティが複数の値を持つ可能性があるため、最初以外の値は角括弧でインデックス指定することで参照できる。例えば、以下の架空のBII引数を考える：

```json
{
  "menuItem": {
    "@context": "http://schema.org",
    "@type": "MenuItem",
    "name": "pizza",
    "menuAddOn": {
      "@type": "MenuItem",
      "name": "pineapple"
    },
    "menuAddOn": {
      "@type": "MenuItem",
      "name": "extra cheese"
    }
  },
  "date": "2000-01-01"
}
```

この例では、パス `"menuItem.menuAddOn[1].name"` は `"extra cheese"` と評価される。

角括弧がない場合、最初の値が想定される。`[0]` を使用することはノーオペレーションである。すなわち、`"menuItem.menuAddOn.name"` と `"menuItem.menuAddOn[0].name"` は同等であり、_どちらも_ `"pineapple"` と評価される。

## BII引数から共通型へのマッピング

BII引数マッピングをサポートするGoogle標準APIは、少なくとも以下のSchema.orgエンティティ型からActions on Google共通型への型マッピングをサポートする**べきである**（should）：

| 共通型            | Schema.orgエンティティ型                                      |
| ----------------- | ------------------------------------------------------------ |
| [Date][]          | [Date][schema-date]                                          |
| [DateTime][]      | [DateTime][schema-datetime]                                  |
| [TimeOfDay][]     | [Time][schema-time]                                          |
| [Duration][]      | [Duration][schema-duration]                                  |
| [Money][]         | [MonetaryAmount][schema-monetaryamount]                      |
| [PostalAddress][] | [Place][schema-place], [PostalAddress][schema-postaladdress] |

## BII引数からJSON型へのマッピング

BII引数マッピングをサポートするGoogle標準APIは、少なくとも以下のSchema.orgエンティティ型からJSON型への型マッピングをサポートする**べきである**（should）：

| JSON型    | Schema.orgエンティティ型   |
| --------- | ------------------------- |
| `string`  | [Text][schema-text]       |
| `number`  | [Number][schema-number]   |
| `bool`    | [Boolean][schema-boolean] |

## BII引数から文字列へのマッピング

[App Actions][]のようなURIベースの履行APIは、BII引数を文字列（URIパラメータ）にのみマッピングできる。これらのAPIは、少なくとも以下のSchema.orgエンティティ型をURIパラメータにマッピングすることをサポートする**べきである**（should）：

- [Text][schema-text]
- [Number][schema-number]
- [Boolean][schema-boolean]
- [Date][schema-date]
- [DateTime][schema-datetime]
- [Time][schema-time]
